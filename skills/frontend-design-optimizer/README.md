# Frontend Design Optimizer - 前端设计优化专家

> 一份让 AI Agent 直接读懂的"前端 UI/UX 优化操作手册"

## 这是什么

这是一个 **AI Skill**，专门帮助开发者将普通前端页面升级为**产品级高品质界面**。

覆盖 8 大维度：

| 维度 | 核心关注点 |
|------|-----------|
| 视觉层次 | 3 秒理解页面主次 |
| 排版系统 | 字号/行高/字重体系化 |
| 色彩系统 | 60-30-10 法则 + Token 化 |
| 响应式布局 | Mobile First + 断点体系 |
| 交互体验 | 即时反馈 + 状态管理 |
| 动画过渡 | 时间曲线 + GPU 加速 |
| 无障碍访问 | WCAG AA 标准 |
| 性能优化 | LCP/FID/CLS 指标 |

## 如何使用

### 方式一：上传到 Skill Center（推荐）

1. 将本目录打包为 zip
2. 上传到 https://cvmops.woa.com/#/skillCenter
3. AI 对话时说："阅读 Frontend Design Optimizer skill，帮我审查这个页面"

### 方式二：直接使用 SKILL.md

把 `skill.md` 的内容粘贴给任意 AI Agent（CodeBuddy / Claude / ChatGPT），它会按里面的规范帮你分析和优化代码。

## 适用人群

- **前端开发者**：快速建立设计审美和最佳实践
- **全栈工程师**：不需要设计背景也能做出好看界面
- **团队 Lead / Reviewer**：作为 UI 层面的 Code Review 检查清单
- **独立开发者**：一个人也能产出产品级 UI
